## 🚀 Installation

1. Clone the repository.

```bash
git clone https://github.com/your-username/sample-project.git
```

2. Move into the project directory.

```bash
cd sample-project
```

3. Install the required dependencies.

```bash
pip install -r requirements.txt
```

4. Run the project.

```bash
python main.py
```
